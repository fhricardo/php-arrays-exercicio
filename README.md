## Exercício: Arrays  
### Crie uma array multidimensional em PHP com as seguintes características:

- Nome do Produto;
- Descrição completa;
- Preço;
- Foto;

O array deve ser salvo em um arquivo com o nome "produtos.php", dentro da pasta "includes"
